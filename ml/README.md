# SaludCopilot — Modelo Predictivo

Random Forest para predicción de tiempos de espera por área.

## Setup

```bash
cd ml
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Estructura

- data/     → datasets (no se suben al repo)
- notebooks/ → exploración y entrenamiento
- src/      → pipeline productivo
- tests/    → validación del modelo
